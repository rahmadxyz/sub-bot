# sub-bot
Free Points Collector for Everyone.


### Installation

Requires [PHP](https://php.net/) v7 to run.

Install Command

```sh
$ git clone https://github.com/rahmadxyz/sub-bot
$ cd sub-bot
```
### Configuration
Open your SUB4SUB APP.
Login using ur google account/contains yotube channel

### How To Run

```sh
$ php sub-bot.php
```
